# The Death Grip (死亡之握) - Android App

**The Death Grip** 是一個有趣的 Android 應用程式，模擬了某些智慧型手機在被以特定方式握持時會出現的訊號丟失問題（通常被稱為「死亡之握」）。

這個專案是對 2010 年左右著名的天線門事件的一個幽默致敬。

## 功能

*   **互動式訊號模擬**：按住螢幕上的「死亡區域」（手機邊框左下角），可以看到訊號格數迅速下降，模擬訊號丟失的情況。
*   **Bumper 保護殼模式**：點擊 Home 鍵可以選擇啟用「Bumper」保護殼模式。啟用後，訊號表現會變得不穩定（波動）而不是直接掉到零，模擬加裝保護殼後的效果。
*   **音效回饋**：包含訊號變化和按鍵操作的趣味音效。
*   **懷舊體驗**：帶您重溫早期智慧型手機的經典爭議話題。

## 如何建置 (Build)

這是一個使用 Gradle 的標準 Android 應用程式專案。

1.  **前置需求**：Android Studio 和 JDK 1.8+。
2.  **開啟專案**：在 Android Studio 中開啟此專案目錄。
3.  **同步**：等待 Gradle 同步依賴項目。
4.  **執行**：在模擬器或 Android 實體裝置上建置並執行 App。
